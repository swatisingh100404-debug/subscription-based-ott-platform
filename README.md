# Subscription-Based OTT Platform

A full-stack OTT streaming platform built using the MERN stack.

## Features

- User Authentication (JWT)
- Subscription Management
- Video Streaming
- Watchlist
- Admin Dashboard
- Secure Payment Integration
- Responsive UI

## Tech Stack

### Frontend
- React.js
- Vite
- Tailwind CSS

### Backend
- Node.js
- Express.js
- MongoDB

## Installation

### Backend
cd backend
npm install
npm start

### Frontend
cd frontend
npm install
npm run dev
